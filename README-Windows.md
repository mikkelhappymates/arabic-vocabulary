# عربي Vocabulary - Windows

A native desktop application for learning Arabic vocabulary with Danish and English translations.

## System Requirements

- Windows 10 or later (64-bit)
- ~50 MB disk space

## Installation

1. Download `ArabicVocabulary-0.1-beta-Setup.exe` from the [Releases page](https://github.com/mikkelhappymates/arabic-vocabulary/releases)
2. Run the installer
3. If Windows SmartScreen appears, click **"More info"** → **"Run anyway"** (the app is not code-signed)
4. Follow the installation wizard
5. Launch from the Start Menu or Desktop shortcut

## Data Storage

Your vocabulary is saved locally at:
```
C:\Users\<YourName>\Documents\ArabicVocabulary\vocabulary.json
```

This folder is created automatically on first launch.

## Features

- **Arabic Word Storage** - Store Arabic words with transliteration, English, and Danish translations
- **Built-in Arabic Keyboard** - Virtual keyboard with full Arabic alphabet and diacritics
- **Tag System** - Organize words with custom tags
- **Search & Filter** - Find words quickly
- **Dark Mode** - Arabic-inspired dark theme with geometric pattern

## Uninstalling

1. Open **Settings** → **Apps** → **Installed apps**
2. Find "Arabic Vocabulary" and click **Uninstall**

Or use the uninstaller in the Start Menu folder.

> **Note:** Your vocabulary data in Documents will NOT be deleted during uninstall.

## Troubleshooting

### App won't start
- Make sure you have Windows 10 or later
- Try running as Administrator

### Arabic text not displaying correctly
- The app uses system fonts; Arabic should display correctly on all modern Windows versions

### Antivirus warning
- Some antivirus software may flag unsigned applications
- Add an exception for the app if needed
